# 🕌 NurDouha

**NurDouha** is an open-source Islamic platform that provides a modern and lightweight way to read the **Quran** and access Islamic knowledge tools.

The project currently includes a **Quran reader web application** with verse search and Mushaf reading mode.
Future modules will include **Prayer Times, Hadith collections, and Islamic books**.

---

## ✨ Features

* 📖 **Quran Reader** with clean Arabic text
* 🔎 **Verse Search** across the Quran
* 📜 **Mushaf Mode** (continuous Quran page layout)
* 📱 **Responsive Design** for mobile and desktop
* ⚡ **Lightweight JSON API**
* 🧭 Planned modules: Prayer times, Hadith, Islamic books

---

## 🖥 Web Application

Open the Quran reader:

```text
web-app/reader.html
```

Or deploy the project using GitHub Pages.
