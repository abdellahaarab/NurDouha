Introduction

NurDouha is an open-source Islamic platform that provides tools for Quran reading, prayer times, Qibla direction, and more.

Getting Started

Clone the repository:

git clone https://github.com/abdellahaarab/NurDouha.git

Open the project:

cd NurDouha

Run the app:

open index.html

Features

  • Quran Reader
  • Verse Search
  • Mushaf Mode
  • Prayer Times
  • Qibla Direction
  • Responsive UI
